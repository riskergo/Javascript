Operatornya :

  • ( && )

    • kedua kondisi harus true agar menghasilkan true
  • ( || )

    • cukup salah satu kondisi yang true agar menghasilkan true
  • ( ! )

    • pembalikan nilai boolean (true/false) dari dari nilai aslinya

Operator-operator ini akan menghasilkan nilai boolean
